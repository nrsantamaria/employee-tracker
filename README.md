# Employee Tracker

#### By _**Nicole Santamaria** and **Sowmya Dinavahi**_

## Description
An app to track employees and their projects.

### Prerequisites

* Web browser with ES6 compatibility
Examples: Chrome, Safari
* Ruby
* Bundler

### Installing

* Clone repository: `$ git clone https://github.com/nrsantamaria/employee-tracker`
* Navigate to folder: `$ cd employee-tracker`
* Install gems: `$ bundle install`
* In a separate tab start postgres: `$ postgres`
* Set up the database: `$ rake db:create`
* Create tables: `$ rake db:migrate db:test:prepare`
* Start server: `$ ruby app.rb`
* In your browser visit: `localhost:4567`
